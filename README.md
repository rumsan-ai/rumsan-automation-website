# Rumsan Automation UI

Rumsan Automations where production-ready AI assistants and business workflows come to life. From chatbots and document intelligence to voice automation and internal tools, everything is unified in one platform to help teams launch, test, and run practical automations across communication, HR, finance, and community operations.

## Features

- **🎙️ VoxFlow** - A voice workflow platform for conversational interactions, multilingual audio transformation, and guided automation.
- **📋 Community Tool** - A machine learning powered data utility for cleaning, normalizing, and fuzzy matching community records into structured, high-confidence datasets.
- **📄 CV Evaluation & Scoring** - A resume screening product that extracts candidate data, applies AI-based scoring, and generates hiring-ready summaries.
- **🎙️ Open Source Voice Agent** - A self-hosted voice assistant stack with customizable prompts, logic, and deployment control.
- **🤖 AskBhunte AI Chatbot** - A Facebook Messenger assistant that delivers real-time market insights, news updates, and automated user support.
- **📅 Employee Leave Management** - An HR workflow tool that captures leave requests, standardizes approvals, and keeps leave records organized.
- **🧾 Invoice Validation Portal** - A finance operations portal for invoice intake, validation, issue tracking, and claim submission workflows.
- **💬 Discord Chatbot Integration** - A community chatbot solution for automated moderation, instant responses, and engagement at scale.
- **📨 Telegram Chatbot Integration** - A messaging automation bot for customer queries, support flows, and rapid broadcast communication.
- **📧 Job Application Email Organizer** - An email automation workflow that classifies applications, filters candidates, and streamlines recruiter follow-up.


## Quick Start

1. **Install dependencies**

   ```bash
   pnpm install
   ```

2. **Set up environment**

   ```bash
   cp .env.example .env
   ```

   Then update `.env` with your real values.

3. **Run development server**

   ```bash
   pnpm dev
   ```

4. **Open app**

   Go to `http://localhost:3000`

## Tech Stack

- Next.js 16 + TypeScript
- Tailwind CSS + Shadcn/UI
- N8N webhook integration
